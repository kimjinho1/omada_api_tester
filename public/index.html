API Parameter Reference

Radio - channelWidth
"1"=20MHz, "2"=Auto 20/40MHz, "4"=40MHz, "6"=80MHz/160MHz, "8"=160MHz (6G)

Radio - channel
"0"=Auto
2.4G: "1","2","3"..."13"
5G: "36","40","44","48","149","153","157","161","165" ...
6G: "1","5","9"..."233"

Radio - txPower
2.4G: 4~25 dBm, 5G: 4~25 dBm, 6G: 4~23 dBm
실제 범위는 모델별 상이 (deviceMisc.minPower/maxPower 참조)

Radio - wirelessMode
-2=Auto
2.4G: b/g/n/ax/be mixed
5G: a/n/ac/ax/be mixed
6G: ax/be mixed
SSID - band (bitmask)
1=2.4G, 2=5G, 3=2.4+5G, 4=6G, 5=2.4+6G, 6=5+6G, 7=All

SSID - security
0=Open, 2=WPA-Enterprise, 3=WPA-Personal, 4=PPSK w/o RADIUS, 5=PPSK w/ RADIUS

pskSetting - versionPsk
1=WPA-PSK, 2=WPA2-PSK, 3=WPA/WPA2-PSK, 4=WPA3-SAE

pskSetting - encryptionPsk
1=Auto, 3=AES (WPA3-SAE는 AES 필수)

SSID - pmfMode
1=Mandatory, 2=Capable, 3=Disable

SSID - deviceType
1=EAP only, 2=Gateway only, 3=EAP+Gateway

Device - status
1=Connected, 2=Disconnected
