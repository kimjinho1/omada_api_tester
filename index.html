Wireless Setup Workflow

Radio → SSID → WLAN Group 지정 순서로 무선 설정을 진행합니다. 각 단계에서 설정 후 검증까지 수행됩니다.

Step 1. 현재 상태 조회

Radio Config
조회 버튼을 클릭하세요
SSID List (현재 WLAN Group)
조회 버튼을 클릭하세요

Step 2. Radio 설정 (AP별)

변경할 밴드만 체크하세요. 체크하지 않은 밴드는 변경하지 않습니다.

⚠ 5G 채널/대역폭 변경 시 channelRange(주파수 리스트)가 자동 포함됩니다. (supportChannelRange 모델: EAP653 UR 등)

Step 3. SSID 생성/수정 (WLAN Group 단위)

상단에서 선택한 WLAN Group에 SSID를 생성/수정/삭제합니다.

Step 4. WLAN Group → AP 지정 (내부API)

상단에서 선택한 WLAN Group을 Target AP에 지정합니다.

API Parameter Reference

Radio - channelWidth (대역폭)
"2" = 20MHz
"3" = 40MHz
"4" = Auto (2G)
"5" = 80MHz
"6" = Auto (5G)
"7" = 160MHz
미지원값 → 가장 가까운 값으로 자동변환

Radio - channel (채널)
"0" = Auto
2.4G (Auto): 1~11
5G 20MHz: 36~165 (25ch)
5G 40MHz: 36~161 (24ch, 165 제외)
5G 80MHz: 36~161 (19ch, 128~144 제외)
5G 160MHz: 36~128 (16ch)
대역폭에 따라 선택 가능 채널이 다름
5G GET 응답은 channelRange 인덱스일 수 있음
실제 채널은 /radios의 wp5g.actualChannel 확인

Radio - wirelessMode
-2 = Auto (권장)

Radio - txPower (dBm)
2.4G: 7~23 | 5G: 7~23 | 6G: 7~23
모델별 상이 (deviceMisc.min/maxPower)

Radio - boolean
radioEnable: 무선 ON/OFF
channelLimitEnable: 채널 제한 (5G/6G)

⚠ 5G channelRange
5G 채널/대역폭 변경 시 channelRange 필수
없으면 Success 반환되나 미반영
SSID - band (bitmask)
1 = 2.4G
2 = 5G
3 = 2.4G + 5G
4 = 6G
5 = 2.4G + 6G
6 = 5G + 6G
7 = 전체

SSID - security (인증)
0 = Open
2 = WPA-Enterprise
3 = WPA-Personal (PSK)
4 = PPSK w/o RADIUS
5 = PPSK w/ RADIUS

pskSetting - versionPsk
1 = WPA-PSK
2 = WPA2-PSK (권장)
3 = WPA/WPA2-PSK (호환)
4 = WPA3-SAE (AES+PMF필수)

pskSetting - encryptionPsk
1 = Auto (TKIP+AES)
3 = AES only (WPA3 필수)
SSID - pmfMode (PMF)
1 = Mandatory (WPA3 필수)
2 = Capable (권장)
3 = Disable

SSID - deviceType
1 = EAP only
2 = Gateway only
3 = EAP + Gateway

SSID - broadcast
true = SSID 표시
false = SSID 숨김

SSID - boolean 필드
mloEnable: MLO
enable11r: 802.11r 로밍
hidePwd: 비밀번호 숨김
vlanEnable: VLAN 사용
guestNetEnable: 게스트
greEnable: EoGRE 터널
gikRekeyPskEnable: 그룹키 갱신

Device - status
1 = Connected
2 = Disconnected
